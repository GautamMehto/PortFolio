A web designer creates the layout and design of a website. In simple terms, a website designer makes a site look good. They use design programmes to create visual elements and interface (UI), which means they strategically design a site that's intuitive and easy for visitors to navigate

UI/UX Designer

UI/UX design refers to the combined discipline of User Interface (UI) design and User Experience (UX) design. While related, these are distinct aspects of the design process, each focusing on different aspects of creating a digital product, such as a website or a mobile application.

Front-End Developer

A front-end developer is a type of web developer who focuses on the client side of web development. In other words, front-end developers are responsible for building the user interface (UI) and user experience (UX) of a website or web application. They work with the visual and interactive elements that users directly interact with on their browsers.
